# 🥐 Bakery App

A simple front-end project for a bakery website, built using only **HTML** and **CSS**.

---

## 🎯 Project Objective

To practice and apply knowledge of:

- Semantic HTML
- CSS Fundamentals
- `inline-block` and `float`
- CSS `transform: translate`
- Layering with `z-index`
